# You Matter Jewelry Landing Page

A landing page for **You Matter Jewelry** — a handmade jewelry brand focused on one-of-a-kind pieces, emotional meaning, and strong product presentation.

## About the brand

You Matter Jewelry creates original handmade jewelry pieces designed as small personal statements.  
Each piece is unique, crafted with care, and built around the idea that the person wearing it matters.

## Purpose of this project

This repository contains the public landing page for the brand.  
Its goal is to present the visual identity, product philosophy, and online presence of You Matter Jewelry in a clean and focused way.
